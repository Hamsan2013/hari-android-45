# Hari

Android WebView application that loads [https://visual-melody.lovable.app](https://visual-melody.lovable.app).

## Features
- WebView with JavaScript & DOM storage enabled
- Pull-to-refresh support
- AdMob Banner & Interstitial ads
- Splash screen with app logo
- JavaScript interface for native features

## Build

### Android Studio
1. Open this project in Android Studio
2. Build > Build Bundle(s) / APK(s) > Build APK(s)

### GitHub Actions
Push to GitHub and the workflow will automatically build a debug APK.
Download it from the Actions tab > Artifacts.

## Configuration
- Package: `com.example.myapp`
- Target SDK: 34
- Min SDK: 21
